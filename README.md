# LeetCode Problem Tracker

A lightweight Chrome Extension that allows users to bookmark LeetCode problems directly from the problem page and manage them via a quick access popup menu.

## Features
* ➕ Injects a custom tracking button directly into LeetCode's React UI.
* 💾 Saves problems locally using the Chrome Storage API.
* 🚀 Quick launch saved problems in new tabs.

## Installation
1. Clone this repository or download the ZIP.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the project folder.
